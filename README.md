# 🏦 Bank System

A fully-featured banking system built from scratch using **C++ with Object-Oriented Programming (OOP)** principles. This project aims to simulate a real-world banking application with all the core functionalities you'd expect from an actual bank.

## 📌 About

This project is designed to replicate the operations of a real banking system — from client management and account handling to transactions, user authentication, and administrative controls. Every feature is being carefully crafted following clean architecture and layered design patterns.

The system is built with a **daily incremental development** approach, where new features and improvements are added consistently over time.

## 🏗️ Architecture

The project follows a **layered OOP architecture** for clean separation of concerns:

```
Bank System
├── Models/          → Data models (Client, Person, User, Admin)
├── Repository/      → Data persistence and file I/O
├── Services/        → Business logic and operations
├── UI/              → Console-based user interface screens
└── Utility/         → Helper classes (String, Date, Validation)
```

## 🎯 Planned Features

- 👤 Client account management (Create, Read, Update, Delete)
- 💰 Deposits, Withdrawals, and Transfers
- 🔐 User authentication and role-based access control
- 👨‍💼 Admin panel for system management
- 📋 Transaction logging and history
- 🔍 Search and filtering capabilities
- 📊 Account balance reports
- 🛡️ PIN code security and validation
- 📁 Persistent data storage

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/AHMEDMOM1/Bank.git
   ```
2. Open `oopBankProjectAgain.sln` in **Visual Studio 2022**
3. Build the solution (`Ctrl+Shift+B`)
4. Run without debugging (`Ctrl+F5`)
